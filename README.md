# Quiz Game 

This is a simple Python quiz game project made by a beginner learner.

##  Features:
- 20 MCQ questions (Physics + Biology)
- Score system
- Final result evaluation
- Simple text-based interface

##  How to run:
```bash
python quiz.py

##  How to play:
1.Run the program using Python
2.Read each question carefully
3.Choose your answer (a, b, or c)
4.Your score will be shown at the end
5.Try to get the highest score!

##  Score System:
20/20 → Excellent
17+ → Very Good
12+ → Good
Below 12 → Need improvement
